# Changelog

Все значимые изменения в проекте «Сбор» (sbor.team) будут документироваться в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/).

## [Unreleased]

### Added
- **Создание тренировок** — новая страница `/trainings/create` для планирования тренировок с выбором команды, даты, времени, места проведения и типа тренировки
- **Создание матчей** — новая страница `/matches/create` для создания товарищеских и турнирных матчей с автодополнением соперников
- **Модальное окно создания событий** — на дашборде добавлено модальное окно для быстрого выбора типа события (объявление, тренировка, матч)
- **Управление командой** — на странице профиля команды добавлены кнопки редактирования и удаления с подтверждением
- **Загрузка логотипа команды** — в форме редактирования команды добавлена возможность загрузки логотипа через FileService
- **Приглашения по ролям** — кнопки «Пригласить игрока/тренера» теперь передают параметр роли для предзаполнения модального окна приглашения
- **Новый порядок вкладок** — на странице команды вкладки переупорядочены: Расписание → Турниры → Тренеры → Состав
- **Отдельный блок объявлений** — объявления команды вынесены из вкладок в отдельный блок под вкладками
- **Кликабельные названия команд** — в списке команд названия стали ссылками на профиль команды

### Changed
- **ENUM team_gender** — значения изменены с `male/female` на `boys/girls/mixed` для соответствия спортивной терминологии

### Fixed
- **Nginx 413 error** — добавлена конфигурация `client_max_body_size 10M` для поддержки загрузки файлов
