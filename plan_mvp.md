# План запуска MVP проекта «Детская лига»

## Цель MVP
Обеспечить минимально необходимый функционал для полноценного управления спортивным клубом, командами, тренировками и матчами. Веб-интерфейс доступен **всем ролям** (администратор, тренер, родитель, игрок). Мобильное приложение — дополнительный канал доступа.

## Принципы MVP
- **Web-first**: веб-версия покрывает 100% функций для всех ролей
- **Адаптивный дизайн**: веб-сайт работает на мобильных устройствах
- **API-first**: Backend API обслуживает и веб, и мобильное приложение
- **Модульная архитектура**: nwidart/laravel-modules для изоляции доменов

### Фаза 1: Ядро системы — Клубы и Пользователи (6 недель)
**Срок:** ~42 дня

**API и Авторизация:**
- Подготовка OpenAPI контрактов
- Реализация авторизации (Email, OAuth Google/Apple) через Laravel Sanctum.
- JWT-токены, refresh-токены, сброс пароля.
- RBAC-middleware для разграничения доступа по ролям.
- REST API: /auth/*, /clubs/*, /teams/*, /players/*, /coaches/*.

**Веб-интерфейс (все роли):**
- Регистрация, логин, OAuth, активация по ссылке-приглашению.
- Онбординг клуба (админ): создание клуба → команды → тренеры → игроки.
- Дашборд адаптивный по роли (админ/тренер: управление; родитель/игрок: расписание).
- Управление клубом и командами (админ/тренер).
- Просмотр состава команды, карточки игроков и тренеров (все роли).

**Управление пользователями:**
- CRUD для игроков, тренеров, родителей с профилями.
- Связь родитель↔ребёнок через user_parent_player.
- Загрузка фото через FileService → S3 → таблица files.
- Импорт игроков из Excel/CSV.

**Уведомления:**
- Email-приглашения тренерам и родителям.
- Базовая настройка Push (FCM/APNs) и Telegram Bot.

**Результат:** Клуб укомплектован. Все роли могут войти через веб.

### Фаза 2: Тренировки (5 недель)
**Срок:** ~35 дней

**API:**
- Расписание тренировок: разовые и регулярные (шаблоны с днями недели).
- Посещаемость: RSVP от родителей (подтверждение/причина), отметки тренером.
- Авто-продление отсутствия (травма + срок).
- CRUD мест проведения (venues).

**Автоматизация:**
- Cron: авто-генерация тренировок из шаблонов (recurring_trainings).
- Job: авто-проставление отсутствия на будущие тренировки при травме.

**Веб-интерфейс (все роли):**
- Календарь тренировок: переключение месяц/список (прошедшие серым, предстоящие ярко).
- Карточка тренировки: дата, время, место, тренер, список участников.
- RSVP через веб для родителей: подтверждение / причина отсутствия.
- Просмотр расписания для игроков (read-only).
- Управление тренировками: создание, изменение, отмена (админ/тренер).

**Уведомления:**
- Push + Telegram: новая тренировка, изменение, отмена.

**Результат:** Полноценный модуль тренировок для всех ролей через веб.

---

### Фаза 3: Матчи и Турниры (6 недель)
**Срок:** ~42 дня

**API:**
- Три типа турниров: товарищеский матч, однодневный, регулярный.
- Создание матчей, формирование составов, RSVP от родителей.
- Live-матч: таймер, фиксация событий (голы, карточки, замены), пересчёт счёта.
- Турнирная сетка: групповой этап, playoff.

**Веб-интерфейс (все роли):**
- Карточка матча: соперник, место, состав, счёт (live для всех ролей).
- Турнирная сетка и таблица результатов.
- RSVP на матч через веб (родители).
- Live-экран матча: таймер, лента событий, текущий счёт (родители/игроки).
- Управление матчем: запуск таймера, фиксация событий (админ/тренер).

**Уведомления:**
- Push: матч создан, напоминание за 24ч, гол в реальном времени, итог матча.

**Результат:** Полноценное ведение матчей и турниров через веб для всех ролей.

### Фаза 4: Статистика, уведомления и экспорт (3 недели)
**Срок:** ~21 день

**API и интерфейс:**
- Статистика по игрокам: голы, ассисты, посещаемость, игровое время.
- Статистика по командам: результаты матчей, процент посещаемости.
- Фильтрация по периоду (сезон, 3 месяца, всё время).
- Страница статистики с графиками (все роли через веб).
- Экспорт отчётов в Excel.
- Интеграция расписания с Google/Apple Calendar через iCal.
- Страница уведомлений: список, настройки включения/отключения по типу (все роли).

**Результат:** Готовая веб-версия MVP для всех ролей.

---

### Фаза 5: Мобильное приложение (8 недель)
**Срок:** ~56 дней

**Архитектура:**
- Проектирование (React Native / Flutter), настройка Auth + Deep Links.
- API Client (REST + WebSocket), Offline Sync Layer (SQLite/Hive).

**Экраны:**
- Главный экран, профиль, настройки.
- Календарь тренировок + RSVP.
- Календарь матчей + RSVP + Live-экран.
- Статистика игрока, профиль ребёнка.
- Уведомления с deep-link навигацией.

**Инфраструктура:**
- Push FCM/APNs + Offline-кэш.
- QA-тестирование, публикация в App Store и Google Play.

**Результат:** Запуск мобильного приложения v1.0.
