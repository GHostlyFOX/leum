# Postman Collection

## Импорт коллекции

1. Откройте Postman
2. Нажмите **File → Import**
3. Выберите файл `Сбор_API_Collection.json`

## Настройка переменных

В коллекции настроены переменные:

| Переменная | Значение по умолчанию | Описание |
|------------|----------------------|----------|
| `base_url` | `http://localhost:8000/api/v1` | Базовый URL API |
| `access_token` | *(пусто)* | Bearer токен авторизации |
| `refresh_token` | *(пусто)* | Refresh токен |

## Использование

1. Выполните запрос **Auth → Вход**
2. Скопируйте `access_token` из ответа
3. В коллекции нажмите на вкладку **Variables**
4. Вставьте токен в поле `access_token`
5. Теперь все запросы будут автоматически использовать токен

## Структура коллекции

```
Детская лига (Сбор) API
├── Auth              # Регистрация, вход, выход
├── Clubs             # CRUD клубов
├── Teams             # Команды и состав
├── Trainings         # Тренировки, посещаемость
├── Matches           # Матчи и события
└── References        # Публичные справочники
```

## Pre-request Scripts

Для автоматического обновления токена можно добавить Pre-request Script:

```javascript
// Проверка и обновление токена перед запросом
if (pm.environment.get("access_token") === "") {
    // Выполнить запрос на обновление токена
    // или перенаправить на страницу логина
}
```
