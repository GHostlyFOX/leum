## Рекомендации по использованию

### Зелёно-чёрная схема

| Использование | Рекомендуемый вариант | Тема |
|--------------|----------------------|------|
| **Favicon** | V1 | Светлая |
| **App icon iOS** | V1 Dark | Тёмная |
| **App icon Android** | V2 или V3 | Светлая |
| **Шапка сайта** | V4 | Светлая |
| **Загрузочный экран** | V2 Dark | Тёмная |
| **Печать (визитки)** | V3 или V5 | Светлая |
| **Соцсети (аватар)** | V1 | Светлая |

### Где какая версия
- **Светлая версия** → для белых/светлых фонов (сайт, документы)
- **Тёмная версия** → для тёмной темы, загрузочных экранов, ночного режима

---

## SVG формат
Все логотипы в векторном формате SVG — можно масштабировать до любого размера без потери качества.
